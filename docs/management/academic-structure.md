# Academic Structure

Universities, domains, departments, and specializations form the organizational hierarchy.

## Universities

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/v1/universities` | List all |
| POST | `/api/v1/universities` | Create |
| GET | `/api/v1/universities/:id` | Get one |
| PUT | `/api/v1/universities/:id` | Update |
| DELETE | `/api/v1/universities/:id` | Delete |
